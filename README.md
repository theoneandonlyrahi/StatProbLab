# StatProbLab

An interactive statistics and probability learning tool built for students and educators. StatProbLab provides a clean, modern alternative to tools like Stapplet, with a focus on visualization, simulation, and hands-on exploration of statistical concepts.

Live Site: https://theoneandonlyrahi.github.io/StatProbLab/
(might be blocked by your IT; contact them to whitelist it)

## Features

### Categorical Data Analysis
- One Variable Single Group: bar charts, segmented bar charts, pie charts
- One Variable Multiple Groups: grouped and stacked bar charts
- Two Variable Categorical Data: side-by-side charts, segmented bar charts, mosaic plots

### Quantitative Data Analysis
- One Variable Single Group: histograms, box plots, summary statistics
- One Variable Multiple Groups: comparative histograms and box plots
- Two Variable: scatter plots and correlation analysis

### Distributions
- Normal
- Uniform
- Geometric
- Binomial,
- Poisson

### Simulations
- Monte Carlo Pi Estimation
- Dice Roll
- Random Walk
- Coin Flip
- Spinner
- Card Draw
- Same Birthday Problem
- Statistical Significance
- Parkinson's Smell Test

## Usage

Navigate using the sidebar on the left. Click any card to open the corresponding tool. Each tool has an input panel, visualization output, and stats summary.

Note: Currently desktop-only. Mobile support is not yet implemented.
